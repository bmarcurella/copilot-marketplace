# Privacy

The plugins and skills in this repository are content-only packages (instructions, manifests, and
templates). They do not collect, store, or transmit personal data themselves.

- **Cowork plugins** run inside Microsoft 365 Copilot / Cowork; data handling is governed by your
  Microsoft 365 tenant's policies and the [Microsoft Privacy Statement](https://privacy.microsoft.com/privacystatement).
- **MCP connectors** referenced by a plugin (e.g., the Microsoft Learn MCP server) send your queries
  to that service; see the respective service's privacy terms.
- **CLI plugins** run locally through GitHub Copilot CLI / VS Code and are governed by the
  [GitHub Privacy Statement](https://docs.github.com/site-policy/privacy-policies/github-privacy-statement).

Questions: open an issue on this repository.
